# Основний логотип / Main Logo

## Варіанти логотипу

### 1. Повноколірний / Full Color

Основна версія логотипу для використання на світлих фонах.

- Фіолетовий текст "APPLIED SCIENCES FACULTY"
- Зелена крапка-акцент
- Декоративна хвиляста лінія з хрестиком зверху

### 2. Монохромний / Monochrome

Сіра версія логотипу для чорно-білого друку або нейтральних макетів.

- Всі елементи в сірих тонах
- Підходить для офіційних документів

### 3. Для темного фону / For Dark Background

Інвертована версія для використання на темних (зокрема фіолетових) фонах.

- Білий текст
- Зелена крапка-акцент залишається
- Біла декоративна лінія зверху

---

## Елементи логотипу

```
     ✚
  ~~~◡~~~
  ━━━━━━━
  APPLIED
  SCIENCES
  FACULTY●
```

- **Хрестик (✚)** — символ зв'язку з УКУ
- **Хвиляста лінія** — символ динаміки та інновацій
- **Пряма лінія** — стабільність та структурованість
- **Зелена крапка (●)** — акцент, що виділяє факультет

---

## Завантаження

Для завантаження файлів логотипу зверніться до оригінального брендбуку.
